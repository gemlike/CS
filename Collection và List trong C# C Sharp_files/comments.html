FindAll có cho phép nhận một tham số kiểu Predicate<type>, đây chính là delegate trả về true/false. Nên bạn có thể viết biểu thức lambda ngay ở tham số. Muốn viết rõ hơn thì có thể viết:

Predicate<Product> findpro_100 = (product) => {
return product.Price > 100;
};

List<Product> p_100 = products.FindAll(findpro_100);
